# CSE Value & Trend Screener 🇱🇰

A web-based platform to identify undervalued shares in the **Colombo Stock Exchange** using Fundamental and Technical Analysis.

## Features
- **Golden Cross Detection:** Identifies when the 50-day SMA crosses above the 200-day SMA.
- **Pre-Cross Alerts:** Detects narrowing gaps between averages to find trends *before* they break out.
- **Value Filtering:** Filters by P/E and P/B ratios to ensure long-term safety.
- **Interactive Charts:** Powered by Plotly for deep-dive analysis.

## How to Run
1. Clone this repo: `git clone https://github.com/YOUR_USERNAME/CSE-Screener.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
